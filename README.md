# SheepSweeper
Clone of MineSweeper game written in C++ and Qt5 originaly
forked here:
https://github.com/katecpp/sheep_sweeper

- Now work with Qt5
- Add high Scores
- 6 levels of difficulty (Chuck Noris is the max)
- Custom difficulty (no high score in custom mode)
